# KCET Tracker

While preparing for KCET, one thing kept bothering me:

Some days I could solve problems quickly and accurately, and other days the same questions suddenly felt slow and stressful. I had no clear way to measure consistency or know whether I could actually perform under pressure.

I had already built Airion before this, but since it was CLI-based, it eventually started feeling too rigid and inconvenient for daily use during intense preparation.

So I built **KCET Tracker** — a more intuitive and flexible system focused entirely on tracking preparation, performance, and revision.

The app was designed around simplicity and focus:

* dark theme UI
* minimal distractions
* separate tabs for study, checking, stats, and revision
* fast workflows so tracking never became annoying

Users could:

* log what they were studying
* track sessions and practice sets
* enter answers during sessions
* mark guessed questions separately
* verify scores later using answer keys
* analyze performance and revision patterns over time

One of the most satisfying parts was being able to actually *see* progress visually. That feedback loop made studying feel less chaotic and much more motivating.

The project was heavily vibe-coded using Antigravity because my priority at the time was simple:

> build it fast, use it immediately, and get back to studying.

And honestly, it worked.

I ended up scoring a 19k rank in KCET, and this project became one of the tools that genuinely helped me stay consistent throughout preparation.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/kcet-tracker.git
```

Navigate into the project directory:

```bash
cd kcet-tracker
```

Install dependencies:

```bash
npm install
```

Build for production:

```bash
npm run build
```
Start the development server:

```bash
npm run dev
```
